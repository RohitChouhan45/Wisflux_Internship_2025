Project Overview

This project is a front-end clone of YouTube, designed to replicate the user interface and core features of the popular video-sharing platform. Built with modern front-end technologies, the clone allows users to:

# Browse videos: A dynamic video feed displays popular and recommended videos, mimicking the layout and style of YouTube.

# Search functionality: Users can search for videos by title, tags, or channels through a responsive and intuitive search bar.

# Watch videos: Click to view videos in a media player with controls for play, pause, volume, and full-screen mode.

# Video thumbnails and previews: Each video has a clickable thumbnail that leads to the individual video page.

# Responsive design: The platform features a fully responsive layout, optimized for both mobile and desktop devices.

# Navigation: Includes a side menu for easy access to different sections like trending, subscriptions, and the user profile page.
